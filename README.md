# 🕸️ Scrape-It

A small project that takes a URL (either a website or LinkedIn profile) and tries to pull out useful information from it.

Nothing fancy — just learning how scraping works.

---

## 💡 What this does

- You give it a link  
- It reads the page  
- It extracts some basic info  

That’s it.

---

## 🔗 Works with

- Normal websites URL  
- LinkedIn profiles *(limited)*  (URL) 

---

## 📦 What it can extract

### 🌐 Websites
- Title  
- Headings  
- Some text  
- Links  

### 💼 LinkedIn (basic)
- Name  
- Headline  
- About section *(if available)*
